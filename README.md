# 🍿 netflix-content-based-recommendation-system

This is a simple content-based movie recommendation system built using the Netflix dataset and **cosine similarity**.

## 🎯 Objective
To recommend similar movies to users based on movie content features like:
- Genre
- Description
- Cast
- Crew

## 🗂️ Dataset
The dataset used contains metadata of Netflix movies (you can specify the file name if needed).

## ⚙️ Tools & Libraries Used
- Python
- Pandas
- Scikit-learn
- Numpy
- Jupyter Notebook

## 📓 Notebook
All logic and implementation are in:
- `Netflix_Movie_Recommender.ipynb`

## 💡 Key Highlights
- Uses **cosine similarity** to compute similarity between movie feature vectors.
- Displays the top recommended movies similar to the selected one.
